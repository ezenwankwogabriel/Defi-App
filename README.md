# DEFI APP

Create and deploy a DEFI APP

## Install

This repository contains Smart Contract code in Solidity (using Truffle), tests (also using Truffle)

To install, download or clone the repo, then:

`npm install`
`truffle compile`
`truffle migrate`

To run script to initialize tokens, stake myToken and withdraw token, then:

`truffle exec ./getMyTokenBalance.js`

## TEST APP

To run truffle tests:

`truffle test ./test`
